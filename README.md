# ⏳ Kairos: The Cognitive Orchestrator

Kairos is a high-performance study orchestrator built for Arch Linux (Hyprland) and Windows. It is designed to move beyond simple time-tracking and instead manage the biological and psychological state of the user to conquer executive dysfunction.

## 🚀 Core Features
- **AI Task Decomposition:** Turn overwhelming goals into frictionless "Tiny Steps."
- **Cognitive Load Budgeting:** Energy-aware scheduling to prevent burnout.
- **Hyprland Synergy:** Deep integration with your tiling window manager.
- **Discord Rich Presence:** Let the world know you're in a state of Flow.
- **Evolutionary Design:** Continuous R&D via automated research loops.

## 🛠 Tech Stack
- **Backend:** Rust + Tauri (userspace-networking for remote admin).
- **Frontend:** React + TypeScript + Tailwind CSS (Glassmorphic Design).
- **IPC:** Hyprland `hyprctl`.
- **Presence:** Discord RPC.
